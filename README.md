# 1. Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/collabspace.git
cd collabspace
```

# 2. Setup MongoDB Atlas
1. Create cluster  
2. Get connection string  
3. Allow 0.0.0.0/0  
4. Replace <password>  

# 3. Backend Setup
```bash
cd backend
npm install
```

Create `.env`:
```env
PORT=5000
MONGODB_URI=your_string
JWT_SECRET=your_secret
JWT_EXPIRE=7d
FRONTEND_URL=http://localhost:3000
```

Run backend:
```bash
npm run dev
```

# 4. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

Create `.env`:
```env
VITE_API_URL=http://localhost:5000/api
```
